🏠 What is PocketHQ?

PocketHQ is a workspace you build yourself. Instead of a one-size-fits-all app, you add the tools you actually need — called blocks — and connect them with automations that do things for you automatically.

Think of it like building your own business dashboard, without needing to know how to code.

⬜ Blocks — Your Tools

Every block is a different type of tool. Click + Add Block on the Home screen to add one.

🗃
Database
Like a spreadsheet — but smarter. You define the columns (Name, Phone, Status, Date, etc.) and add rows of data. Has a table view and a Kanban board view if you add a Status column. Use it for clients, jobs, orders, tenants, leads — anything you track in rows.
✅
Task List
Your to-do list. Add tasks with due dates and priority levels. Overdue tasks show in red. Filter by Today, Overdue, or High Priority. You can get a notification reminder when tasks are due — enable this in Settings.
📈
Tracker
Log numbers over time and see a running total. Good for revenue, units sold, hours worked, miles driven, or anything else you measure. Just hit + Log Entry, enter the number, and it adds up.
📝
Notes
A freeform text area for anything that doesn't fit elsewhere. SOPs, business info, scripts, checklists, reminders. Just type and hit Save.
💰
Budget
Track money coming in (income) and going out (expenses). Every entry shows your net profit automatically. Use categories like "Materials", "Labor", "Sales" to stay organized. Filter to see only income or only expenses.
📅
Schedule
Add appointments, meetings, deadlines, or any events with a date and optional time. Today's events are highlighted. Mark events done when they're over. Filter by Upcoming, Today, or Past.
☑
Checklist
A repeatable step-by-step list. Great for end-of-day routines, job site safety checks, onboarding steps, or any process you repeat. Check off items as you go. Hit ↺ Reset to start fresh for the next round.
🗃 Using the Database — Kanban Board

If your Database has a column set to type Select (like a Status column), a Kanban button will appear at the top. Click it to switch to a card board view where each column is a stage.

You can drag and drop cards between columns to move a record from one stage to another — for example, dragging a job from "In Progress" to "Done".

To go back to the table view, click ☰ Table.

⚡ Automations — Make Things Happen Automatically

Automations watch for something to happen, then do something for you. Every automation has three parts:

1
Trigger — The thing that sets it off. Example: "When a job's Status changes to Done."
2
Conditions (optional) — Extra filters. Example: "Only if the Value is over 500."
3
Actions — What happens. Example: "Send me a notification" + "Create a task called Send Invoice."
Example automation: When a tenant's Lease End date = today → Send notification "Lease ending for {Name} today" → Create task "Contact {Name} about renewal" due in 3 days.

You can add multiple actions to one trigger. Use {Field Name} in notification text to insert real data — like {Name} or {Status}.

Use the ▶ Run button on any automation to test it manually.

🔔 Notifications

PocketHQ uses your browser's built-in notifications. To enable them:

1
Go to Settings in the left sidebar.
2
Click Enable Notifications and allow when your browser asks.
On mobile (iPhone or Android): First add this page to your home screen (tap the Share button → "Add to Home Screen"). Then open it from your home screen icon and enable notifications from Settings. Notifications only work when the app is installed this way on mobile.

Once enabled, you'll get notified when tasks are overdue, due today, or when any automation fires a notification action.

💾 Your Data

All your data is stored on your device — in your browser's local storage. Nothing is sent to a server. This means:

  • It works offline once loaded.
  • Clearing your browser data will erase it — use Export Data in Settings to back it up regularly.
  • It doesn't automatically sync across devices. Back up on one device and restore on another using the export file.
